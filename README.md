# DigiArtz — KOE
> Art Has No Limits

A digital art portfolio and gallery platform built by KOE.
Features character illustrations, landscapes, vehicles, and sketches.

## 🌐 Live Site
https://digiartz.net

## ✨ Features
- Dynamic gallery with Supabase backend
- Subscription tiers (Lite / Premium / Max)
- Website Templates section
- Comment system
- Admin mode

## 🛠 Built With
- HTML / CSS / JavaScript (single-file)
- Supabase (database + storage)
- Cloudflare Pages (hosting)
